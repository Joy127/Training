數位科技職群

培訓具備與時俱進數位科技應用能力之人才

數位圖文傳播

訓練概要

緣由：國際印刷科技與圖像傳播工作環境迅速變遷，造成產業人力斷層，加上文創產業興起，圖文傳播職能工作需求倍增。資訊技術、產品企劃與專案撰寫、品牌包裝與設計、圖文整合工作流程與色彩管理作業等，期能藉由視覺、美學、創意、創新的圖像理論與職能結合，投入產業工作以增強我國就業率及產業動能，提供我國對本專業職類，參加本職類之專業訓練

訓練目標

  1. 教導學員能熟悉數位印刷與資訊科技整合生產概論、圖文傳播資訊技術、印前製作實務管理、印前製作色彩管理實務、成本分析與估價、包裝科技至文化創意商品開發等相關專業知識。
  2. 培訓受訓學員熟練業務估價、材料成本分析、色彩管理應用、印前設計、UV噴墨印刷設計及印前工作流程管理專業技能，包裝設計與實務展現，書籍、海報製作實務呈現、電子書規劃、課程配合專家指導，結訓後能自行規劃展場佈置從事圖文傳播數位化工作。
  3. 將職場經驗與工作態度融入課程，使受訓學員能有職業道德，企業倫理，以陶冶其品德，有助於未來與企業銜接。
  4. 輔導考照：印前製程乙級技術士檢定。

就業展望

  • 能獲得與本科相關之證照及工作機會。

課程介紹

  1. 攝影技術-攝影原理、數位相機基本介紹、紀實攝影與商業攝影、光圈、快門、感光度介紹、棚內(外)攝影實務、白平衡及攝影技巧實務、攝影鑑賞與應用實務。
  2. 圖文傳播資訊技術-"印刷基本知識、各式印刷版原理與特性、製版、印刷、裝訂、後加工流程、數位印刷發展現況。
  3. 互動電子書製作-Indesign多媒體素材介紹、版面規劃、版型處理與檔案格式規劃、電子書排版實務製作教學、實作整合應用。
  4. 色彩管理-色彩管理與檢測概論、色彩描述檔製作與輸出、色彩管理與檢測設備、顯示器及輸出校色、i1/ iO設備操作。
  5. 多媒材設計與應用-多媒材種類與材料介紹、數位印刷機、雷射雕刻機、包裝平台切割機設備操作、商品圖像設計與生產製作、設備使用與維護。
  6. 印刷品質管理與業務經營-印刷的紙張與油墨適性、綠色印刷與環保、印刷品與檢測、印刷業務開發與報價、印刷內部成本與材料分析、印刷業務國際報價流程（估價、協調）、紙張、印工、製版、裝訂價格估價、國內外實務估價練習、業界參訪。
  7. 數位行銷-從設計談行銷、企畫書寫作、行銷廣告設計實務、數位媒體應用實務。

數位影音設計與行銷

訓練概要

緣由：近年來隨著數位科技的蓬勃發展，因應網路興盛與智慧型手機普及，行銷方 式更是多元，而文字、圖片、聲音及影像更是宣傳時必備元素，能傳遞所欲傳達的訊息。利用文字寫出讓人有感的文案是成功關鍵之一，而精美的圖片則更能第一時間吸引視覺，若輔以影片來進行溝通，則效果更佳。因為多數的人會被精美圖片所吸引，會喜歡影片生動的閱聽方式，故企業無不抓住此商機，結合文字、圖片、聲音及影像，發揮創意與趣味性，透過簡單的圖片編輯與影音剪輯來博得青睞。加上若能進一步拍攝照片、製作優化圖片、拍攝影片，可以完成基本的影像作品，更是加分。本分署數位媒體科技應用職群，包含數位圖文傳播科技班、數位影音設計與行銷班等，期透過職群模式規劃整合文、圖、聲、影以達綜效。

訓練目標

  1. 培訓受訓學員能瞭解影片製作流程、腳本、分鏡、攝影知識、故事行銷等相關專業知識。
  2. 培訓受訓學員能熟悉與運用平面攝影、動態攝影、影音編輯、數位應用、電腦繪圖影像設計，並透過分組專題製作進行實務操作。
  3. 將職場經驗與工作態度融入課程，使受訓學員能有職業道德，企業倫理，以陶冶其品德，有助於未來與企業銜接。
  4. 輔導證照試題練習：攝影丙級。

就業展望

  • 與本科相關之證照輔導，並能至相關職能場所工作。

課程介紹

  1. 平面攝影-基礎平面攝影、攝影美學與構圖、色溫與色調及顏色、燈光之運用與實作等
  2. 動態攝影-影片製作流程、腳本、分鏡、影片拍攝作業與運用等
  3. 影音編輯-數位影音製作認識、影片後製作業、剪輯軟體操作及其相關應用
  4. 數位應用-各式行銷媒材種類介紹與應用、各式輔助設備運用等
  5. 資訊傳播與行銷應用-資訊傳播與行銷介紹 、故事行銷、簡報與表達
  6. 電腦繪圖影像設計-點陣式影像設計、向量式電腦繪圖、影像構圖版面規劃
  7. 傳播資訊技術實務應用-攝影丙級技術士檢定試題練習
  8. 綜合應用練習-綜合應用練習、分組練習
  9. 專題製作-影音專案分組製作
  10. 輔導參加攝影丙級技術士檢定

PHP資料庫網頁設計

訓練概要

緣由：現今網路已經和生活密不可分，許多的服務都向網路及雲端發展，因此可以發現市場對於相關的程式設計人材需求大增。現有網頁設計領域也由原本的視覺設計推向動態互動及後端整合，除了原本的繪圖軟體的操作，漸次的往HTML5、CSS、JavaScript、jQuery及PHP資料庫的多項技術整合，同時也要了解各項程式框架的應用，才能有效率的進行開發，因此熟悉這些技術都是目前業界所需的必備技能。但是不論學習哪些技術，最重要的還是必須設計出一個好用、吸引人的網站，擁有前後端網頁整合的技術以及動態圖像表達能力，才能更具就業競爭力。

訓練目標

  1. 使受訓學員能熟悉網際網路應用有關、網頁設計、資料庫規劃、程式設計等相關專業知識。
  2. 使受訓學員熟練網頁美編基礎、動態資料庫程式設計專業技能，於結訓後能從事網站規劃程式設計師、資料庫規劃開發，網頁圖案繪製及美工排版等相關行業之工作。
  3. 使受訓學員能有職業道德，企業倫理，以陶冶其品德。
  4. 輔導考照：網頁設計乙級技術士。

就業展望

  1. 平面設計師
  2. 美術編輯人員
  3. 網頁視覺設計師
  4. UI介面設計人員
  5. 網站程式設計師
  6. PHP資料庫規劃開發人員

課程介紹

  1. 網頁排版編輯-HTML、CSS、Dreamweaver
  2. 視覺影像設計-Photoshop、illustrator
  3. 數位媒體應用-視覺傳達設計、UI/UX流程
  4. 網頁動態技術-JavaScript、JQuery、Bootstrap
  5. 資料庫程式設計-PHP、XAMPP
  6. 資訊網路概論-資訊架構及網路應用概論
  7. 網頁設計實務-網頁設計乙級檢定術科輔導

網路工程師實務應用

訓練概要

緣由：資訊不斷演進，惟均須透過網路以達到資源共享、資料傳遞等需求，各行各業對於網路依賴越來越高，如何運用網際網路資源，了解網路建置規劃、伺服器管理等相關知識，並使能短時間內能充分學習道如何規劃建置企業所需網路架構及伺服器(網站)建置管理等，以收事半功倍的效果，並能提供失業者參加本職頪之訓練。

訓練目標

  1. 使受訓學員能熟悉網際網路應用有關網路規劃概論、伺服器規劃應用、虛擬化技術應用等相關專業知識。
  2. 使受訓學員熟練網路規劃應用、伺服器管理應用、架設網站及雲端(虛擬化)伺服器管理專業技能，於結訓後能從事企業MIS系統(維護)工程師、硬體維護人員、雲端伺服器(網站)之應用系統維護等相關行業之工作。
  3. 使受訓學員能有職業道德，企業倫理，以陶冶其品德。
  4. 輔導考照：輔導學員參與網路架設乙級檢定考試。

就業展望

  1. ISP網際網路網站及網路建置及管理人員。
  2. 企業內部網路及網站之建置、維護、管理。
  3. 電腦系統維護、資訊設備管制人員。
  4. 固網及行動業界之網路工程人員。
  5. 網路架設規劃、系統建置網路環境管理。

課程介紹

  1. 電腦概論-電腦軟體與硬體、數字系統與資料表示法、認識作業系統、電腦軟體應用丙級術科實作
  2. 電腦硬體維修-電腦拆裝、BIOS啟動設定、作業系統安裝、多重開機作業系統安裝、作業系統備份及還原、工具隨身碟製作、電腦維修實作、電腦硬體裝修丙級術科實作
  3. 網路規劃與管理-網路基本運作認識、網路模型運作、ICMP與ARP協定、網際網路協定IPv4規劃、Cisco IOS介紹與常用設定、IP路由協定設定、Switch設定原理、廣域及區域網路協定設定、網路管理工具、CCNA認證
  4. Windows伺服器建置與管理-Server安裝、Server基本環境設定、使用者管理與AD、本地與網路檔案管理、列印伺服器管理、群組原則與安全設定、網頁伺服器安裝設定、FTP檔案伺服器、SMTP/POP郵件伺服器、DNS伺服器
  5. Linux 建置管理-認識Linux、系統操作、系統安裝、系統管理、套件管理、指令操作、檔案管理、使用者帳號管理、網路指令與管理、網頁伺服器架設、檔案伺服器架設、名稱伺服器架設、郵件伺服器架設、DHCP伺服器架設與管理
  6. 綜合應用練習-無線網路設備架設、光纖接續、路由器設定、網頁伺服器架設及檔案傳輸伺服器架設等專題應用練習
  7. 檢定應用實務應用-網路架設乙級技術士技能檢定學科、術科練習

前端網頁開發技術

訓練概要

緣由：近年網頁設計的趨勢，都廣泛使用前端網頁技術開發出互動性極佳的網站，加上行動載具的快速發展，帶動了跨平台網頁設計的潮流，前端網頁設計師成為企業熱門的徵才對象。要成為一位前端網頁設計師，包含了許多的工具軟體與技術的學習，從Photoshop、Illustrator數位影像的運用到HTML、CSS、JavaScript、jQuery網頁程式語言及API資料串接應用，都是目前業界所需的必備技能。但是不論學習哪些技術，最重要的還是必須設計出一個好用、吸引人的網站，擁有RWD網站設計能力以及動態圖像表達能力，才能更具就業競爭力。

訓練目標

  1. 使受訓學員能熟悉前端網頁開發技術等相關專業知識。
  2. 使受訓學員熟練前端網頁開發技術等專業技能，於結訓後能從事前端網頁工程師、前端視覺設計師、UI視覺設計師及平面設計人員等相關行業之工作
  3. 使受訓學員能有職業道德，企業倫理，以陶冶其品德。
  4. 輔導考照：網頁設計丙級技術士。

就業展望

  1. 平面設計人員
  2. UI視覺設計師
  3. 網頁視覺設計師
  4. 網頁前端工程師

課程介紹

  1. 網頁排版編輯-HTML5、CSS3、Bootstrap、RWD響應式網頁製作、網頁編輯器使用
  2. 視覺影像設計-Photoshop數位影像處理、Illustrator向量繪圖設計
  3. 數位媒體應用-視覺傳達設計、UI/UX設計流程
  4. 網頁動態技術-JavaScript、jQuery、API串接應用、 Git版本控制、SPA開發模式介紹、Vue.js前端框架
  5. 資料庫程式設計-NoSQL資料庫操作、MongoDB資料管理、Node.js開發工具
  6. 網頁設計實務(160小時)-網頁設計專題製作
